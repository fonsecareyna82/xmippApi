Get Pyramid Filter. Function: GetPyramidFilter

Purpose: Get the coefficients of the filter (reduce and expand filter) Return the coefficients in g[ng] and in h[nh]

Convention: g[ng] for the reduce filter h[nh] for the expansion filter

Parameters: Filter is the name of the filter

Order is the order for the filters based on splines For the "Spline" filter, Order is 0, 1, 2 or 3 For the "Spline L2" filter, Order is 0, 1, 3 or 5 For the "Centered Spline" filter, Order is 0, 1, 2, 3 or 4 For the "Centered Spline L2" filter, Order is 0, 1, 2, 3 or 4

IsCentered is a return value indicates if the filter is a centered filter TRUE if it is a centered filter FALSE if it is not a centered filter

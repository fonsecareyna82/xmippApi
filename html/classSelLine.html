Detailed Description

Line of a selection file.

The selection file is a collection of (STL list) of Selection Lines. This class needn't be accessed in common programs since the SelFile class offers most of the used functions. However, the class is shown in case you may need to access specifically to any of the Selection Line functions alone. The SelLines can be either comments or data line, they might be of other internal used types (not assigned or to be discarded), but you may assume that if a line is not a comment nor a data line then it can be skipped.

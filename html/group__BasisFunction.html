Detailed Description

This class defines the basis function to use for the reconstruction. Currently, valid basis functions are blobs and voxels.

Create a Zernike Spherical Basis volume. With the same size as an input volume. Z(l,n,m)=R_l^n(r)Y_l^m(x,y,z) for l=0,1,2,..., n<=l, m=-l,...,l The basis volume is created up to Rmax, by default half the size of the input volume

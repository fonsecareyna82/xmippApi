Detailed Description

Tabulated Sinc = SIN(PI*X)/(PI*X)

A lookup-table with the given sampling rate and range is created.

tabsinc TSINC(0.0001,64);
if (TSINC(1)==0)
std::cout << "This is true!\n";

This class is not ported to Python.

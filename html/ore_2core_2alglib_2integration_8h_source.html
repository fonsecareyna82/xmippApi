170 /*************************************************************************
171 Computation of nodes and weights for a Gauss quadrature formula
172 
173 The algorithm generates the N-point Gauss quadrature formula with weight
174 function given by coefficients alpha and beta of a recurrence relation
175 which generates a system of orthogonal polynomials:
176 
177 P-1(x) = 0
178 P0(x) = 1
179 Pn+1(x) = (x-alpha(n))*Pn(x) - beta(n)*Pn-1(x)
180 
181 and zeroth moment Mu0
182 
183 Mu0 = integral(W(x)dx,a,b)
184 
185 INPUT PARAMETERS:
186  Alpha – array[0..N-1], alpha coefficients
187  Beta – array[0..N-1], beta coefficients
188  Zero-indexed element is not used and may be arbitrary.
189  Beta[I]>0.
190  Mu0 – zeroth moment of the weight function.
191  N – number of nodes of the quadrature formula, N>=1
192 
193 OUTPUT PARAMETERS:
194  Info - error code:
195  * -3 internal eigenproblem solver hasn't converged
196  * -2 Beta[i]<=0
197  * -1 incorrect N was passed
198  * 1 OK
199  X - array[0..N-1] - array of quadrature nodes,
200  in ascending order.
201  W - array[0..N-1] - array of quadrature weights.
202 
203  -- ALGLIB --
204  Copyright 2005-2009 by Bochkanov Sergey
205 *************************************************************************/

632 /*************************************************************************
633 Integration on a finite interval [A,B].
634 Integrand have integrable singularities at A/B.
635 
636 F(X) must diverge as "(x-A)^alpha" at A, as "(B-x)^beta" at B, with known
637 alpha/beta (alpha>-1, beta>-1). If alpha/beta are not known, estimates
638 from below can be used (but these estimates should be greater than -1 too).
639 
640 One of alpha/beta variables (or even both alpha/beta) may be equal to 0,
641 which means than function F(x) is non-singular at A/B. Anyway (singular at
642 bounds or not), function F(x) is supposed to be continuous on (A,B).
643 
644 Fast-convergent algorithm based on a Gauss-Kronrod formula is used. Result
645 is calculated with accuracy close to the machine precision.
646 
647 INPUT PARAMETERS:
648  A, B - interval boundaries (A<B, A=B or A>B)
649  Alpha - power-law coefficient of the F(x) at A,
650  Alpha>-1
651  Beta - power-law coefficient of the F(x) at B,
652  Beta>-1
653 
654 OUTPUT PARAMETERS
655  State - structure which stores algorithm state
